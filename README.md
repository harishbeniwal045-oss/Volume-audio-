### AI Hand Gesture Volume Controller

Control your laptop volume with just your hand gestures — no keyboard, no mouse, just pure AI magic ✋🤖

This project allows users to control system volume using real-time hand tracking through a webcam. By measuring the distance between your fingers, the system dynamically increases or decreases volume.

Built using Computer Vision + AI, this project creates a touchless and futuristic user experience.

### Features
1. Real-time hand tracking using webcam
2. Gesture-based volume control
3. Smooth volume increase/decrease
4. Fast and responsive performance
5. Uses AI-based landmark detection
6. Works with most laptops
7. Tech Stack
## Python
OpenCV
MediaPipe
NumPy
Pycaw (for volume control)
### How It Works
1.Webcam captures live video
2.Hand landmarks are detected using MediaPipe
3.Distance between thumb and index finger is calculated
4.Volume is adjusted based on distance
